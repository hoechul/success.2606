**사전 준비**
- **기본 확인**: 기존 통신사 고객센터(휴대폰에서 114)로 전화해 해지 시 발생하는 위약금, 약정 잔여기간, 번호이동(MNP) 예약 가능 여부 확인.
- **MNP 예약번호**: 번호이동 시 기존 통신사에서 발급되는 ‘MNP 예약번호’(유효기간 보통 7일 내외)를 반드시 확보(유효기간은 통신사별 상이).
- **본인확인 수단**: 본인명의 휴대폰, 신분증(주민등록증/운전면허증), 결제수단(신용카드/계좌) 준비.

**필수 제출서류 (대리점에서 수집)**
- **개통/조회 신청서**: `KT 개통/조회 신청서` (필수 서명/기재).
- **개인정보 활용 동의서(1)**: `개인정보 및 개인위치정보 활용 동의서(1)` (첫 번째 동의서).
- **개인정보 활용 동의서(2)**: `개인정보 및 개인위치정보 활용 동의서(2)` (두번째 동의서).
- **신분증 사본**: 고객 신분증 원본 또는 사본 확인.
- **결제정보**: 카드 또는 계좌번호(자동이체 신청 시 필요).
- (선택) **대리인 서류**: 대리 가입 시 위임장 + 대리인 신분증.

**개통 가능 단말기(고객 선택 항목)**
- **지원 단말**: 갤럭시 점프4, S25엣지, S26, S26+, S26 울트라, S26 엣지, 갤럭시 Z 폴드7
- **확인 항목**: 재고/색상/용량, 요금제별 공시지원금·할부 조건, 기기 출고가, 유심(eSIM) 가능 여부, 기기 IMEI 확인.

**PASS 앱 인증(번호이동·신규 공통 필수)**
- **설치·가입**: 고객이 스마트폰에 `PASS` 앱 설치 → 본인 인증(휴대폰 본인인증 + 신분·비밀번호 설정) → 인증서(모바일 인증서) 발급.
- **용도**: 온라인 본인확인, 전자서명 대체, 번호이동·개통 처리 시 본인확인 수단으로 사용.
- **유의사항**: 인증서 발급 전 기존 번호로 본인인증이 필요하므로, 기존 번호(또는 신규 번호 발급 전엔 본인명의 다른 수단) 사용 가능 여부 확인.

**KT 번호이동(개통) 절차 — 순서대로**
1. **사전 준비 완료 확인**: 위약금/약정 확인, MNP 예약번호 확보, 신분증 및 동의서 준비, PASS 인증 완료.
2. **가입정보 수집**: 고객명, 생년월일, 주소, 기존통신사명, MNP 예약번호, 원하는 요금제, 단말기 선택(모델/색상/용량), 결제수단.
3. **서류 작성 및 동의**: `개통/조회 신청서`와 개인정보 동의서(1),(2)에 고객 서명 혹은 전자서명(PASS) 수행.
4. **본인확인·인증**: PASS 앱 인증서로 본인확인 진행(온라인 가입자 인증 필수).
5. **개통 신청 접수**: 대리점 시스템 또는 KT 온라인 개통 페이지에 정보 입력 후 개통 신청(단말 IMEI, 유심 타입 등 기재).
6. **심사 및 승인**: KT에서 신청내용 검증(보통 실시간). 승인되면 MNP 승인 처리.
7. **유심 교체/설정**: 유심 배송 또는 매장 수령 후 유심 장착 및 완료(또는 eSIM 활성화 절차 수행).
8. **서비스 확인**: 통화/데이터 발신·수신 확인, 번호 정상이동 완료 확인.
9. **사후 안내**: 요금제, 청구일, 고객센터 연락처, 개통 관련 문제 시 조치 방법 안내.

**신규가입(번호이동 없이 새번호) 주요 차이**
- MNP 예약번호 불필요.
- PASS 인증은 여전히 본인확인 수단으로 필수.
- 신규번호 발급 → 유심/단말 연결 → 서비스 확인.
