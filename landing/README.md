# KT 공식 인증 대리점 · 모바일 상담 랜딩 페이지

"폰 바꿀 때 됐나요?" — 카카오 대화 느낌의 **모바일 첫화면형** 랜딩 페이지.
신규가입·번호이동 상담(리드)을 **구글폼**으로 받고, 카카오/전화로도 바로 연결됩니다.
빌드 없는 정적 HTML 한 파일이라 Vercel에 그대로 배포됩니다.

```
landing/
├─ index.html   # 랜딩 본체 (CSS/JS 인라인, 외부 의존성 0)
├─ vercel.json  # 정적 배포 설정
└─ README.md    # 이 문서
```

---

## 1) 배포 전 3곳만 교체 (필수)

`index.html` 상단 CONFIG 주석 참고. 에디터 찾기·바꾸기로 처리하면 빠릅니다.

| 항목 | 찾을 값 | 바꿀 내용 |
|---|---|---|
| ① 전화번호(링크) | `tel:0000000000` | `tel:0212345678` (숫자만) |
| ① 전화번호(표시) | `0000-0000` | `02-1234-5678` |
| ② 카카오 채널 | `https://pf.kakao.com/`  (data-kakao) | 본인 카카오톡 채널 주소 |
| ③ 구글폼 | `id="leadForm"` 의 `src=""` | 구글폼 임베드 주소 |

**구글폼 연결법**
1. 구글폼 작성 — 추천 항목: 이름 / 연락처 / 가입유형(신규·번호이동) / 관심 단말 / 희망 요금제 / 개인정보 수집·이용 동의
2. 우측 상단 **보내기 → `< >`(임베드 HTML)** → `<iframe src="...">` 의 **src 값만 복사**
3. `index.html` 의 `id="leadForm"` iframe `src=""` 에 붙여넣기
4. src가 채워지면 폼이 표시되고, 비어 있으면 카카오/전화 대체 박스가 보입니다. 폼 높이는 `.formwrap iframe { min-height }` 로 조절.

푸터의 사업자 정보(상호·대표자·사업자등록번호·주소·영업시간)도 실제 값으로 채워주세요. (단통법·전자상거래 표시)

## 2) Vercel 배포 — 이 저장소 + Root Directory 지정

새 저장소를 만들 필요 없이, 이 저장소를 Import하면서 **Root Directory만 `landing`** 으로 지정하면 됩니다.

1. [vercel.com](https://vercel.com) → **Add New → Project**
2. GitHub 연결 → **`hoechul/SPARTA.RPA`** Import
3. **Root Directory** → `Edit` → **`landing`** 선택
   - Framework Preset은 `Other`(정적)로 자동 인식, 빌드 명령 없음
4. **Production Branch**를 현재 작업 브랜치로 지정하거나, `main`에 머지 후 배포
5. **Deploy** → `*.vercel.app` 주소 발급 → QR/문자/카톡으로 배포

> Root를 `landing` 으로 잡으면 저장소 루트의 다른 프로젝트와 섞이지 않고 이 랜딩만 배포됩니다.

CLI로 이 폴더만 즉시 배포하려면: `npm i -g vercel && cd landing && vercel --prod`

## 3) 전환율(하루 10명) 운영 팁
- 광고/카톡 링크는 `#apply` 로 바로 — 예) `https://.../#apply`
- 구글폼 응답에 **이메일/스프레드시트 알림**을 켜고 신청 즉시 연락 (응답 속도가 전환을 좌우)
- 히어로 문구·혜택 금액을 주 단위로 바꿔 A/B 테스트
- GA4·Meta 픽셀을 `</body>` 직전에 넣으면 캠페인별 전환 측정 가능
